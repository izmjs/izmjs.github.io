## Controller Snippets

### Add controller documentation `doc:ctrl`

```javascript lineNumbers=true
/**
 * $1
 * @controller $2,
 * @param {import('express').Request} req The request
 * @param {import('express').Response} res The response
 * @param {Function} next Go to the next middleware
 */
```

### Add controller documentation `ctrl`

```javascript lineNumbers=true
/**
 * $1
 * @controller $2
 * @param {import('express').Request} req The request
 * @param {import('express').Response} res The response
 * @param {Function} next Go to the next middleware
 */
exports.$3 = async function $3(req, res, next) {
  $4;
};
```

### Response Format `resp`

```javascript lineNumbers=true
{
    success: $1,
    result: {$2}
}
```

```javascript lineNumbers=true
/**
 * @type { IAM.default }
 */
module.exports = {
  prefix: '/',
  routes: [{
    path: '/',
    methods: {
      get: {
        iam: 'modules:moduleName:fileName:apiKey',
        title: '',
        parents: ['modules:moduleName', 'modules:moduleName:fileName'],
        description: '',
        middlewares: [
          
        ],
      },
    },
  }],
};
```

## Module Snippets

### Create a Model `module:model`

```javascript lineNumbers=true
/**
 * Module dependencies.
 */
const { Schema, model } = require('mongoose');

const { timestamps } = require('@config/index').lib.mongoose;

const Schema = new Schema({
  title: String,
}, {
  timestamps,
});

module.exports = model('', Schema);
```

## Postman snippets

All of the bellow snippets are used in a [JSDoc block](https://jsdoc.app/about-getting-started.html#adding-documentation-comments-to-your-code) inside an IAM file

### Add the body `@body`

Will add by default a JSON payload

```javascript lineNumbers=true
/**
 * @body
 * {
 *   "": ""
 * }
 */
```

In order to add a different format of the payload, please use oneof these snippets:

- `@urlencoded` to set the body of an URL encoded payload
- `@formdata` to set the body of an Form Data payload
- `@bodyMode` to set a custom payload format

### Add a test script to the request `@test`

```javascript lineNumbers=true
/**
* @test
* pm.test("Status code is 200", function () {
*   pm.response.to.have.status(200);
* });
*/
```

### Add a prerequest script to the request `@prerequest`

```javascript lineNumbers=true
/**
 * @prerequest
 * pm.environment.set("", "");
 */
```

### Set the params of the request `@params`

```javascript lineNumbers=true
/**
 * @params
 * [{
 *   "key": "",
 *   "value": "",
 *   "description": ""
 * }]
 */
```

### Set the headers of the request `@headers`

```javascript lineNumbers=true
/**
 * @headers
 * {
 *   "": ""
 * }
 */
```

### Add an example to the request `@example`

```
/**
* @example
* {
*   "name": "Example name",
*   "response": {
*     "status": 200,
*     "headers": {
*       "": ""
*     },
*     "body": {
*       "": ""
*     }
*   },
*   "request": {
*     "params": [{
*       "key": "",
*       "value": "",
*       "description": ""
*     }],
*     "headers": {
*       "": ""
*     },
*     "body": {
*       "": ""
*     }
*   }
* }
*/
```

You add as many examples as you need.
